- CLI 入口统一采用 `if __name__ == "__main__": raise SystemExit(main())` 模式，`main()` 返回 0/1/2 表示成功/缺少配置/数据不足。
- 通过 `sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))` 把仓库根加入导入路径，再以 `importlib.util.spec_from_file_location` 动态加载 `apps/quant-*-mcp/*.py`，避免安装依赖即可运行。
- 数据库连接字符串统一用 `_to_psycopg_url` 将 `postgresql+psycopg://` 前缀剥离为 psycopg 原生 URL。
- 受测标的集合（CN_A_EQUITIES / CN_FUNDS / US_EQUITIES）以大写 InstrumentId 常量列表形式集中声明，并在循环中经 `parse_instrument_id` 解析后再传入下游。
- Skill 脚本遵循 fail-closed：遇到空输入、歧义或不可用数据时打印 stderr 并返回非零退出码，由上层 Agent 决定重试或提示用户。
